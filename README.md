# Este repositorio contiene ejercicios, prácticas y ejemplos desarrollados durante el aprendizaje de bases de datos relacionales y NoSQL.

## Se trabajan principalmente dos tecnologías:

- 🐘 PostgreSQL: para el modelado y gestión de bases de datos relacionales, consultas SQL, joins, normalización y operaciones avanzadas.
- 🍃 CouchDB: para el manejo de bases de datos NoSQL orientadas a documentos, consultas con Mango Query, replicación y gestión vía HTTP/REST.

## 🎯 Objetivo del repositorio
- Comprender las diferencias entre bases de datos relacionales y NoSQL.
- Aprender a diseñar y consultar bases de datos eficientes.
- Practicar operaciones CRUD en ambos sistemas.
- Implementar importación, exportación y migración de datos.
- Familiarizarse con herramientas como psql, curl y Fauxton.

## 🛠️ Tecnologías utilizadas
PostgreSQL
Apache CouchDB
SQL
JSON
Bash / Terminal

## 📌 Contenido
Consultas SQL básicas y avanzadas
Modelado de bases de datos
Operaciones CRUD
Exportación e importación de datos
Replicación en CouchDB
Ejercicios prácticos de clase

## 🚀 Notas

Este repositorio está en constante actualización como parte del proceso de aprendizaje en bases de datos.
